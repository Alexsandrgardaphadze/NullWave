# 🎵 NullWave

A personal music collector and organizer desktop app built with C# and Avalonia UI.

## About
NullWave lets you save and organize music from YouTube, Spotify, Instagram, and local files in one unified library.

## Tech Stack
- .NET 8
- Avalonia UI 12
- SQLite + EF Core (coming soon)
- YouTube Data API v3 (coming soon)
- Last.fm API (coming soon)
