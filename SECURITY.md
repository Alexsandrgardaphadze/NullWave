# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Yes    |

## Reporting a Vulnerability
If you find a security vulnerability in NullWave, please do NOT open a public issue.

Instead, contact: open a private GitHub security advisory.

We'll respond within 72 hours and credit you in the changelog if the issue is confirmed.

## Scope
- API key exposure
- Data leakage from local storage
- Malicious URL handling
