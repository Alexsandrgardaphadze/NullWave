## [0.1.0] - 2026-05-26
### Added
- Initial project structure
- Core Track model with source detection
- In-memory LibraryService (add, remove, search, filter)
- MainViewModel with MVVM pattern
- Basic Avalonia UI (dark theme)
- RelayCommand helper
- SourceDetector helper (auto-detects YouTube, Spotify, Instagram)
- Settings window placeholder
