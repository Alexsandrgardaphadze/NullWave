# Contributing to NullWave

Thanks for your interest! Here's how to contribute:

## Rules
- Credit the original author: ZenQuant (PackItPro Team)
- No malicious features (see LICENSE)
- Follow existing code style (.editorconfig)
- Write clean, readable C#

## How to contribute
1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Commit clearly: `git commit -m "feat: add your feature"`
4. Push and open a Pull Request

## Commit style
- `feat:` new feature
- `fix:` bug fix
- `chore:` maintenance
- `docs:` documentation
- `refactor:` code cleanup
